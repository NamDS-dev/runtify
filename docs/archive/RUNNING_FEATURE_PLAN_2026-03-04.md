# 개인 러닝 기능 강화 기획서

> 목표: Nike Running Club 수준의 GPS 경로 추적 + 러닝 요약 구현
> 작성일: 2026-03-04

---

## 현재 러닝 기능 상태

| 기능 | 상태 |
|------|------|
| GPS 거리 측정 | ✅ 구현됨 (Geolocator, 5m 필터) |
| 실시간 시간/페이스 | ✅ 구현됨 |
| BLE 심박수 | ✅ 구현됨 |
| Firestore 저장 | ✅ 구현됨 |
| **GPS 경로 좌표 저장** | ❌ 없음 (좌표를 거리 계산 후 버림) |
| **러닝 중 지도 표시** | ❌ 없음 |
| **러닝 완료 전용 페이지** | ❌ AlertDialog만 있음 |
| **구간별 페이스 기록** | ❌ 없음 |
| **1km 마일스톤 알림** | ❌ 없음 |

## 구현할 기능 (3단계)

---

### Step 1 — 러닝 중 지도 표시

**NRC처럼 달리면서 내 경로가 지도에 그려지는 기능**

#### 화면 레이아웃 변경
```
┌─────────────────────────┐
│                         │
│     지도 (경로 표시)      │  ← 화면 상단 50%
│     현재 위치 마커 🔵    │
│                         │
├─────────────────────────┤
│  1.42 km    08:23       │  ← 거리 + 시간 (크게)
├────────┬────────────────┤
│ 5'52"  │  ❤ 152 bpm    │  ← 페이스 + 심박수
├────────┴────────────────┤
│        ⏹ 정지           │  ← 버튼
└─────────────────────────┘
```

#### 기술 결정: 지도 패키지
→ **flutter_map 확정** (OpenStreetMap, 무료, API 키 불필요)

#### 코드 변경 범위
- `running_page.dart` — 지도 위젯 추가, 경로 좌표 목록 누적
- `pubspec.yaml` — `flutter_map`, `latlong2` 패키지 추가

---

### Step 2 — 러닝 완료 전용 페이지

**현재 AlertDialog → 전체 화면 요약 페이지로 교체**

#### 화면 레이아웃
```
┌─────────────────────────┐
│  🎉 러닝 완료!            │
├─────────────────────────┤
│                         │
│   지도 (전체 경로)        │  ← 시작🟢 ~ 종료🔴 마커
│                         │
├─────────────┬───────────┤
│    거리      │   시간    │  ← 2열 (페이스는 구간별에서만)
│   2.14km    │  12:43   │
├─────────────┴───────────┤
│  ❤ 평균 148bpm  🔥 127kcal │
├─────────────────────────┤
│  +21P 획득!  EXP +21    │  ← 포인트/EXP 강조
├─────────────────────────┤
│  👟 케이던스  평균 182spm  │
├─────────────────────────┤
│  구간별 페이스            │
│  1km  5'32"             │  ← 게이지 바 없음 (100% 기준 불명확)
│  2km  6'15"             │
├─────────────────────────┤
│       [ 홈으로 ]         │
└─────────────────────────┘
```

#### 코드 변경 범위
- `running_result_page.dart` — 신규 파일 생성
- `running_page.dart` — 완료 시 결과 페이지로 이동
- `app_router.dart` — `/running/result` 라우트 추가
- `running_session_model.dart` — `routePoints` 필드 추가

---

### Step 3 — 러닝 기록 상세 화면

**홈에서 기존 러닝 기록 카드를 탭하면 상세 보기**

#### 화면
- Step 2 러닝 완료 화면과 동일한 구조
- 단, 저장된 경로 좌표로 지도 재현
- 삭제 버튼 추가 (선택)

#### 코드 변경 범위
- `running_detail_page.dart` — 신규 파일 생성
- `running_session_card.dart` — 탭 시 상세 페이지로 이동
- `app_router.dart` — `/running/detail/:id` 라우트 추가

## Firestore 데이터 구조 변경

```dart
// running_sessions/{sessionId} 에 필드 추가
{
  ...기존 필드들...,

  // 신규 추가
  "routePoints": [          // GPS 경로 좌표 배열
    {"lat": 37.123, "lng": 127.456, "timestamp": ...},
    ...
  ],
  "splitPaces": [           // 1km 구간별 페이스 (분/km)
    {"km": 1, "pace": 5.53},
    {"km": 2, "pace": 6.25},
  ],
  "region": "서울시 강남구",  // GPS 역지오코딩으로 자동 설정
}
```
